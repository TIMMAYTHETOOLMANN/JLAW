## Evidence-Backed Standards

Every violation MUST include:

### 1. Supporting Evidence
```json
{
  "evidence_type": "exact_quote",
  "source_document": "https://www.sec.gov/Archives/...",
  "source_location": "Page 45, Section 3.2.1, Line 12",
  "exact_content": "THE ACTUAL QUOTE - verbatim",
  "verification_method": "How was this extracted/verified?"
}
```

### 2. Statute Citations
```json
{
  "statute_title": "15 USC § 78p(a)(2)",
  "section": "Section 16(a)(2)(A)",
  "exact_regulatory_text": "THE ACTUAL LAW TEXT - verbatim",
  "violation_type": "Specific violation description",
  "govinfo_url": "https://www.govinfo.gov/..."
}
```

### 3. Reasoning Chain
```json
{
  "reasoning_chain": [
    "1. Transaction executed on 2019-01-18 (extracted from XML)",
    "2. Form 4 filed on 2019-01-22 (SEC EDGAR metadata)",
    "3. Section 16(a) requires filing within 2 business days",
    "4. Calculated business days: 4 days total",
    "5. Exceeds 2-day requirement by 2 business days",
    "6. CONCLUSION: Late filing violation of 15 USC § 78p(a)(2)"
  ]
}
```

### 4. Quality Metrics
```json
{
  "confidence": "high",
  "evidence_strength_score": 0.90,
  "is_reportable": true
}
```
