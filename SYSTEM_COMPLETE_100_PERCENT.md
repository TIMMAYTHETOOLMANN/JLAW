## 🎬 QUICK START

### Initialize System
```python
from src.forensics.enhanced_forensic_system import EnhancedForensicSystem

# Initialize forensic system
system = EnhancedForensicSystem()

# Create new case
case = await system.create_case(
    target="Nike Inc.",
    case_type="sec_filing"
)

# Analyze document
case = await system.analyze_document(
    case,
    document_path="nike_10k_2019.txt",
    document_type="10-K"
)

case = await system.detect_contradictions(case)

# Generate report
report = await system.generate_report(case)
```
