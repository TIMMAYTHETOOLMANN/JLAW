   - Unique case ID
   - Audit trail initialization

2. **Fetches Real SEC Filings**
   - SEC EDGAR API (rate-limited 10 req/sec)
   - All filing types (10-K, 10-Q, 8-K, 4, etc.)
   - Document metadata extraction

3. **Parses Each Filing**
   - HTML/XML/XBRL parsing
   - Table extraction
   - Text analysis
   - Financial data extraction

4. **Detects Violations**
   - Form 4 late filing (2-day deadline)
   - Zero-dollar transactions
   - Material misstatements
   - SOX 302/906 compliance
   - Insider trading patterns

5. **Correlates Legal Authority**
   - Fetches USC statute text from GovInfo API
   - Maps violations to CFR regulations
   - Builds legal citation chains

6. **Generates Reports**
   - DOJ-grade comprehensive report
   - Evidence package (JSON)
   - Chain of custody documentation
   - Statutory citations
