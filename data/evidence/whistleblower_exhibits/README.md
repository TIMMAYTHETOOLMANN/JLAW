# Whistleblower Exhibits

This directory holds exhibits submitted by or on behalf of whistleblowers.

## Confidentiality

All files in this directory are presumptively confidential. Do not share, reproduce, or quote from these materials without authorization.

## Handling Requirements

- Maintain the original file unmodified. Place any annotated version in `processed/` with a reference to the original.
- Record acquisition method and date in the evidence manifest.
- Note whether the exhibit is firsthand observation, documentary evidence, or a combination.

## Analytical Notes

Whistleblower testimony and exhibits represent firsthand accounts. They should be:

- Treated as credible leads requiring independent verification, not as established facts.
- Compared against documentary sources (SEC filings, vendor records, maintenance logs) before inclusion in any report.
- Clearly identified as whistleblower-sourced in any claim verification record.

The available evidence does not establish wrongdoing until corroborated through independent verification.
