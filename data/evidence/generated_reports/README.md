# Generated Reports

This directory holds forensic dossiers and analysis reports produced by the JLAW engine.

## Contents

Reports generated by the JLAW 15-node recursive analysis engine are placed here after a completed run. Each report should be accompanied by its evidence manifest entry.

## File Naming Convention

```
{company}_{analysis_type}_{date_range}_{generated_date}.{ext}
```

Example: `NKE_forensic_dossier_FY2019_FY2020_20240101.pdf`

## Notes

- Generated reports are analytical outputs, not primary source documents.
- Do not treat findings in generated reports as established facts without tracing each finding to a primary source in `raw/` or `sec_filings/`.
- Reports should indicate their confidence level and identify claims that require further verification.
