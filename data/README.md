# Data Directory

This directory holds forensic evidence artifacts and output data generated by the JLAW forensic analysis platform.

## Structure

```
data/
└── evidence/       # Forensic evidence packages and analysis state files
    ├── NKE_Forensic_Analysis_FY2019_FY2020.zip   # NIKE forensic analysis archive
    └── investigation_state.json                   # Saved investigation state
```

## Contents

### `evidence/`

Contains forensic evidence packages produced by the JLAW analysis pipeline:

- **`NKE_Forensic_Analysis_FY2019_FY2020.zip`** — Completed forensic dossier for NIKE, Inc. covering FY2019–FY2020.
- **`investigation_state.json`** — Serialized investigation state for resumable analysis sessions.

## Notes

- Evidence files are generated artifacts and should **not** be modified manually.
- Files in this directory may be subject to chain-of-custody requirements (FRE 902(13)/(14)).
- Analysis outputs from new runs are written to the `output/` directory (git-ignored).
