## Evidence Integrity Verification

### Quick Verification (5 minutes)

**Step 1:** Compute SHA-256 hash of any evidence file
```bash
sha256sum evidence/sec_filings/10-K_2024.pdf
```

**Step 2:** Compare with hash in `evidence/sec_filings/hash_manifest.json`

**Step 3:** If match → Evidence integrity confirmed ✅

### Full Verification (1 hour)

1. Verify all evidence file hashes (triple-hash)
2. Reconstruct Merkle tree from hashes
3. Compare Merkle root with `custody/merkle_tree.json`
4. Validate RFC 3161 timestamp token
5. Review chain of custody for completeness
